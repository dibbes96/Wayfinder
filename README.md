# Wayfinder
Way Finder is a navigation device designed to make orientation simple and intuitive. Instead of using screens or voice prompts, it communicates directions through vibrations: a vibration to the right indicates a right turn, a vibration to the left indicates a left turn.

This system allows the user to fully focus on their surroundings, improving safety and fluidity while traveling. The vibrations are designed to be immediate and easily distinguishable, creating a true tactile language of direction.
